# Feature
- 链接个人博客
- 使用Chrome扩展收藏链接
- 链接点击计数
- 基于标签查询

# Plan
- 链接推荐系统
- 点击习惯统计

# Issue

[Issue Here](https://github.com/everettjf/mostlike.link/issues)

# Based on
- Flask
- MongoDB
- AngularJS
- Redis

# Setup
1. install MongoDB
2. install Redis
3. pip install -r requirements.txt
4. python run.py
